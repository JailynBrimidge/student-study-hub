# DBMS Notes 

## Important Points

1. **Database**
   A database is an organized collection of data that can be stored and accessed easily.

2. **DBMS**
   A DBMS is software that helps users create, manage, and retrieve data from databases.

3. **Primary Key**
   A primary key is a unique identifier for each row in a table.

4. **Foreign Key**
   A foreign key connects one table to another and helps maintain relationships.

5. **Normalization**
   Normalization reduces repeated data and improves database structure.

6. **SQL**
   SQL is the language used to interact with databases.

## Summary

DBMS is important because it helps store information in an organized way and makes data easier to manage securely.

# Computer Networks (CN) Notes

## What is a Computer Network?
A computer network is a group of connected devices that can communicate and share data with each other.

## Important Concepts

1. **Network Topology**
   Topology refers to the layout of a network, such as star, bus, or ring structures.

2. **OSI Model**
   The OSI model has 7 layers that explain how data travels through a network, from physical transmission to application use.

3. **TCP/IP Model**
   This is a simpler model used in real-world networks. It defines how data is sent over the internet.

4. **IP Address**
   An IP address is a unique identifier assigned to each device on a network.

5. **Protocols**
   Protocols are rules that define how data is transmitted. Examples include HTTP, FTP, and TCP.

6. **Bandwidth**
   Bandwidth is the amount of data that can be transmitted over a network in a given time.

7. **Latency**
   Latency is the delay in data transmission between sender and receiver.

8. **Router**
   A router connects different networks and directs data packets to their correct destination.

9. **Switch**
   A switch connects devices within the same network and helps forward data efficiently.

## Summary
Computer networks allow devices to communicate and share information. Understanding networking concepts is important for building and managing modern systems.
